# mp3-media

## Overview

MP3 Media is a ID3 V2 tag editor.

## Command Line Execution

id3 <**path to a mp3 file**>

## Example

id3 src/test/resources/journey.mp3

{"header":{"version":3, "revision":0, "flags":0, "size":300022}, "frames":[{"id":"TIT2", "size":6, "flags":0}]}
